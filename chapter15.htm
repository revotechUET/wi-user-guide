Module 15:  Corporate Database

15.1.     What is Corporate Database

Corporate Database (CODB) is a cloud storage that allows users to store files/folders of their company on i2G platform. Corporate Database will be shared with particular users having permission to access it.

To log into CODB:

Step 1.         Go to https://storage-admin.i2g.cloud/

Step 2.         At the Login dialog, enter your Username and Password.

Figure 151 Log into CODB

Step 3.         Click Login.

Note: If youre having trouble logging in, please contact your administrator or contact us via https://www.i2g.cloud/contact-us/

15.3.     Files you can store in Corporate Database

Any file type can be stored in CODB:

  • General files:
    • Images files: (.JPEG, .PNG, .GIF, .BMP, .TIFF, .SVG)
    • Audio formats: (MP3, MPEG, WAV, .ogg, .opus)
    • Video files: (WebM, .MPEG4, .3GPP, .MOV, .AVI, .MPEGPS, .WMV, .FLV, .ogg)
    • Archive files:  (.ZIP, .RAR, tar, gzip)
    • Markup/code: (.CSS, .HTML, .PHP, .C, .CPP, .H, .HPP, .JS, .java, .py)
    • Text files: (.TXT)
    • Las files: (.LAS)
  • Microsoft Office files:
    • Excel (.CSV, .XLS and .XLSX)
    • PowerPoint (.PPT and .PPTX)
    • Word (.DOC and .DOCX)
    • XML Paper Specification (.XPS)
  • Adobe files: .AI; .PSD; .PDF; .SVG; .TIFF; .TTF

15.4.     Metadata

Each file/folder stored on your CODB has metadata associated with it. Metadata is the information of a files source, size, rating, last modified date, and more. This metadata makes searching and organizing files on storage a lot faster.

When you select a file/folder, its metadata will be displayed in the right bar.

Figure 154 Meta data of an object in Corporate Database

There are 4 types of metadata in CODB:

      General: basic metadata that describes a resource for identification purpose such as File Name, Type, Size, Location, Source, Author, Date Uploaded, Date Modified. These metadata elements are created by automated information processing so users cannot edit it except for the File Name.

      Information: provide information to help users manage resource, describe content and quality of data, track the relationship between files and objects on data tree. There are 6 metadata elements in this class:

  Well, Field, Well Type: the particular resource the file/folder belongs to

  Data Type: type of data in file/folder such as Log, Routine Special Core, MDT, Mud Log

  Quality: quality of file data

  Associate to: view objects on data tree that was associated to this file/folder

      Description: users can summarize information of the file/folder

      More Information: allow users to input any metadata which is relevant or necessary to describe the file/folder.

Click Add Metadata button, the Add Metadata dialog pops up. Then enter a Name and Value, click Ok to add the new metadata for that file/folder.

Figure 155 Add Meta data dialog

15.5.     Upload files and folders to Corporate Database

Step 1.         Click Upload () icon at the top left, Upload Files dialog pops up.

Figure 156 Upload Files dialog

Step 2.         Click Select Files/Folders > choose the files/folder you want to upload then click Open.

Step 3.         Add metadata for items. Keep checking Bulk Edit option if you want to update metadata for multiple items at once, if not, uncheck this one and modify each item manually.

Tip: you can select files/folders many times during one working phase. New selected files/folders will be added right under the previous selected one.

Step 4.         Click Upload after you define all the options.

Note: If you upload a file/folder with the same name, i2G will hold this one. You need to choose Overwrite or Remove from Upload List option. If you want to overwrite all the existing files/folders, click Overwrite All.

15.7.     Search for files/folders

Sometimes remembering precisely where you stored a file might be difficult. I2G allows you to find them more easily by searching for: file name or file metadata.

15.7.1.         Simple search by file name

Step 1.         At the top right, type a word or phrase into the Search box

Figure 157 Simple Search box

Step 2.         Press Enter. Search results will appear as you type

Step 3.         To clear and close the search, click the Close button ( ) next to the Search box or ( ) next to the FILTER: CUSTOM SEARCH MODE

15.7.2.         Advanced search by file metadata

When you cannot find the files or folders you are looking for by using simple Search box, you can perform an advanced search that gives you more options to find them easier.

Step 1.         Click on Advance Search ( ) button next to Search box. The Advance Search dialog pops up.

Figure 158 Advance Search dialog

Step 2.         Choose the kind of item you want to Search for. There are 3 options: File, Folder and All (both File & Folder).

Step 3.         Check/uncheck Sub-folders to include/exclude subfolders in your search.

Step 4.         Select metadata you want to search for File Name, File Type, Author, Data Uploaded, Field, Well, Well Type, Data Type and Custom. Then type in search criteria in the box or select from the available criteria.

Tip: You can combine multiple metadata at once to narrow your search results, for example:

Figure 159 Narrow search results using Advance Search

Note: Custom element is users defined metadata that was added to item earlier.

Step 5.         To clear and close the search, click the Close button ( ) next to the Search box or ( ) next to the FILTER: CUSTOM SEARCH MODE

I2G lets you preview files from within the Corporate Database without having to download them first.

15.9.     Work with Project Storage in CODB window

You can work with any Project Storage directly from CODB window. When you modify any data, the Project Storage from https://wi.i2g.cloud/ will automatically update accordingly.

To work with the Project Storage in CODB window, you need to open a new tab first.

Step 1.         On the top of your CODB window, next to the last tab on the right, click New tab (+) icon.

Step 2.         From the Member list on the left side, drag and drop a project into the new tab. All files and folders of the selected project will be listed here. The name of the tab will be returned to the name of this project.

Figure 1511 Open a Project Storage in CODB window

Tip: You can open as many tabs as you want. You can also view all your tabs and switch among them
